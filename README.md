# Barrage
一个开源的b站直播间弹幕助手，主播可以获取直播间内弹幕信息且可以进行发送弹幕功能。

版本 ***0.0.2***

## 功能介绍

+ 在直播间接收弹幕（已完成，但弹幕有丢失）
+ 在直播间发送弹幕（已完成，错误控制不完善）
+ ...
